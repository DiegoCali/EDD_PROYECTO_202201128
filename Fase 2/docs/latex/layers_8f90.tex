This module contains the definition of the layers\+\_\+tree type, which is a binary tree of layers. Each layer has an id, a number of pixels, a pixel\+\_\+matrix and two pointers to other layers. The module also contains the definition of the pixel\+\_\+matrix type, which is a matrix of pixels. The module also contains the definition of the node\+\_\+layer type, which is a node of a linked list of layers. The module also contains the definition of the queue type, which is a queue of layers.
